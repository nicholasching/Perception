## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Obtain a Gemini API key

   At: https://aistudio.google.com/apikey

3. Create a .env file and add 

   ```bash
    EXPO_PUBLIC_API_KEY = "YOUR_API_KEY"
   ```

4. Start the app

   ```bash
    npx expo start
   ```
